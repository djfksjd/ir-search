# 페이롤리 (합성 평가용 프로젝트)

> 이 폴더는 ir-search 스킬 evals 전용 **합성 fixture**다. 실존 기업·인물과 무관하다.

## 아이템

중소기업 급여·4대보험 정산 자동화 SaaS. 회계 프로그램·은행 API와 연동해
급여명세서 생성, 원천세 신고 파일 작성, 이체 실행까지 한 흐름으로 처리한다.
AI 기반 급여 이상치 탐지(중복 지급·요율 오적용)가 차별점.

## 현재 상태

- 법인 설립 2년차 (서울 강남구 소재)
- 팀: 5명 (개발 3, 사업 2)
- 유료 고객사 40곳, 월 반복매출 발생 중
- 시드 투자 3억 원 유치 완료

## 필요한 것

1. R&D 자금 (이상치 탐지 모델 고도화)
2. 스케일업·글로벌 진출 지원 (일본 시장 검증)
3. 대기업·금융권 연계 프로그램
