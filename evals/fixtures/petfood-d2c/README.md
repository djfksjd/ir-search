# 멍냥키친 (합성 평가용 프로젝트)

> 이 폴더는 ir-search 스킬 evals 전용 **합성 fixture**다. 실존 기업·인물과 무관하다.

## 아이템

반려동물 맞춤 자연식(펫푸드) D2C 구독 서비스. 반려견의 품종·나이·알레르기
정보를 입력하면 주 단위 맞춤 식단을 제조해 정기배송한다. 자체 레시피 엔진과
소규모 HACCP 대응 주방 운영이 핵심.

## 현재 상태

- 사업자등록 전 (예비창업자)
- 팀: 대표 1인 + 파트타임 푸드 스타일리스트 1인
- 프로토타입: 시제품 30가구 베타 배송 완료, 재구매 의향 70%
- 소재지: 대구 (이전 고려: 경북 일부)

## 필요한 것

1. 사업화 자금 (시제품 → 양산 레시피 검증)
2. 식품 제조가 가능한 입주공간 (공유주방/시제품 제작 설비)
3. 멘토링 (식품위생 인허가, D2C 물류)

## 참고

- 대표: 32세 (청년 요건 해당), 대구 거주
- 매출·투자유치 이력 없음
